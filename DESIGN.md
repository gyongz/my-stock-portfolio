# DESIGN.md — 个人持仓 Web 应用

## 产品意象
专业交易终端/券商 App 的 Web 精简版。暗色金融风，数据面板密集但不拥挤，像 Bloomberg Terminal 与东方财富的融合。

## 气质与意象
金融级专业感、数据驱动、沉稳可信。配色以深色背景为基底，绿涨红跌的经典中国股市配色，信息层级清晰。

## 视觉策略
- 黑白灰主色调 + 红绿标识涨跌，不添加多余色彩
- 卡片式布局配合细微阴影，数据区域以分隔线划分
- 图表区域占视觉主体，持仓列表为辅助信息面板

## 配色方案
- **背景**: `#0f172a` (深蓝黑) — 终端感
- **卡片/面板**: `#1e293b` — 次级背景
- **上涨/盈利**: `#22c55e` (绿色)
- **下跌/亏损**: `#ef4444` (红色)
- **主文字**: `#f8fafc`
- **次要文字**: `#94a3b8`
- **边框/分割线**: `#334155`

## 字体排版
- 中文: PingFang SC / Microsoft YaHei
- 数字: 等宽字体 (monospace) 确保数字对齐
- 价格/涨跌幅使用加粗大号字体突出

## 布局结构
```
┌──────────────────────────────────────────────┐
│  总资产 | 总盈亏 | 日盈亏    [工具栏: 导入/导出] │
├──────────────────────────────┬────────────────┤
│                              │                │
│    KLine 图表 (主区域)        │  持仓列表       │
│    - 时间周期切换              │  (可滚动)       │
│    - 技术指标叠加              │                │
│    - 缩放/平移                │                │
│                              │                │
├──────────────────────────────┴────────────────┤
│  底部: 选中持仓的详细信息/盈亏分析              │
└──────────────────────────────────────────────┘
```

## 动效与交互
- hover 状态: 轻微高亮 (背景 `bg-opacity-10`)
- 点击/切换: 瞬时响应，无延迟动画
- 图表拖拽: 流畅平移
- 弹窗: 简洁 fade-in

## 设计禁忌
- 禁用过于鲜艳/花哨的颜色（红绿以外）
- 禁用圆润/卡通风格（所有边角保持直角或极细微圆角）
- 禁用在数据面板使用渐变
- 禁止使用 emoji 图标（全部使用 Lucide 线性图标）

## 技术指标
- 默认支持: MA (均线), MACD, KDJ, RSI, BOLL
- 时间周期: 日线, 周线, 月线, 60分钟, 30分钟
