# DESIGN.md — 个人持仓 Web 应用

## 产品意象
Apple 风格的暗色财经应用。简洁通透，数据面板有呼吸感，像 iOS 股票 App 的 Web 专业版。

## 气质与意象
清爽通透、数据驱动、极致简约。以系统原生质感为基底，信息层级通过间距和模糊分层来实现，而非边框分隔。

## 视觉策略
- 暗色背景，无边框设计，依赖间距分隔内容层级
- 卡片/面板使用 `backdrop-blur` 毛玻璃效果形成视觉分层
- 红涨绿跌（同比于当前价格方向），绿 `#30d158` / 红 `#ff453a`（Apple 标准色）
- 所有圆角统一使用 `rounded-xl` (~12px)

## 配色方案
- **背景**: `#1c1c1e` — Apple 系统暗灰
- **卡片/面板**: `#2c2c2e` — 次级背景
- **上涨/盈利**: `#30d158` (Apple Green)
- **下跌/亏损**: `#ff453a` (Apple Red)
- **主文字**: `#ffffff`
- **次要文字**: `#98989d` (Apple Secondary Label)
- **面板/毛玻璃**: `rgba(44,44,46,0.8)` + `backdrop-blur-xl`

## 字体排版
- 字体堆栈: `-apple-system, BlinkMacSystemFont, "SF Pro Display", "PingFang SC", sans-serif`
- 数字使用等宽字体 (tabular-nums)
- 价格/涨跌幅加粗突出
- 字体大小层级: `text-xs`(标签) / `text-sm`(辅助) / `text-base`(正文) / `text-xl~2xl`(数字)

## 布局结构
```
┌──────────────────────────────────────────────────┐
│  总资产 | 总盈亏 | 日盈亏    [导入/导出/刷新]      │
├──────────────────────────────┬───────────────────┤
│                              │                   │
│   KLine 图表                 │  持仓列表          │
│   (有背板模糊分层)             │  (无边框, 间距分隔) │
│                              │                   │
├──────────────────────────────┴───────────────────┤
│  选中持仓详情                                     │
└──────────────────────────────────────────────────┘
```

## 动效与交互
- hover 状态: 背景 `hover:bg-white/5`，平滑过渡 150ms
- 点击/切换: 瞬时响应
- 弹窗: 简洁 fade-in + scale 动画
- 滚动条: 自定义细滚动条

## 设计禁忌
- 禁用边框/分割线（用间距替代）
- 禁用阴影（用模糊分层替代）
- 禁用过于花哨的颜色（红绿以外）
- 禁用 emoji 图标（全部使用 Lucide 线性图标）
- 禁用直角（全部统一圆角）

## 技术指标
- 默认支持: MA, MACD, KDJ, RSI, BOLL
- 时间周期: 日线, 周线, 月线, 60分钟, 30分钟
